# 码上生成 - 在线QR码工具

## 网站简介
“码上生成”是一款轻量、高效的在线二维码生成工具。用户只需输入文本或链接，即可瞬间生成高清二维码，并支持自定义颜色与尺寸。项目全程采用纯前端技术构建，无需后端服务与数据库，即开即用，保障用户数据隐私零留存。

## 页面与功能
本项目共包含 **8** 个页面，提供从生成到管理的完整闭环体验：
- **首页**：产品核心价值展示与快捷入口。
- **基础生成页**：输入内容，一键生成标准二维码。
- **高级定制页**：支持自定义前景/背景色、码点样式及尺寸调整。
- **批量生成页**：支持多段文本批量导出，提升工作效率。
- **历史记录页**：基于本地存储，便捷查看过往生成记录。
- **使用指南页**：图文并茂的操作说明与场景示例。
- **关于我们页**：项目背景与工具愿景介绍。
- **404 页**：友好的路径错误重定向提示。

## 多 AI 协作与验收
本项目是多 AI 智能体高效协作的实践成果。在开发流程中，各 AI 发挥专长：由主导 AI 负责产品架构与 UI 交互设计，编码 AI 专项攻克 Canvas 绘制与响应式布局，审查 AI 则负责代码规范检测与逻辑漏洞排查。在最终验收阶段，AI 交叉模拟多端用户操作路径，对 8 个页面的生成逻辑、样式适配与本地存储进行全链路测试，确保交付产物零报错、交互流畅，实现严格的自动化质量把控。

## GitHub Pages 部署与访问
本项目依托 GitHub Pages 实现零成本静态部署，入口文件为根目录下的 `index.html`。由于项目无任何后端依赖，只需将代码推送至 GitHub 仓库，并在 Settings 中开启 Pages 服务，即可秒级上线。用户可通过 `https://<username>.github.io/<repo>/index.html` 直接访问，享受极速、稳定的在线二维码生成体验。
