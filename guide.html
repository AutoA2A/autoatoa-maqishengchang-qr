使用指南

以下是码上生成的完整使用教程，帮助您快速上手并了解 QR 码的基本知识。

快速开始

  1. 打开 首页，输入您想编码的文本或网址。
  2. 调整尺寸、颜色、容错率等参数（可选）。
  3. 点击「生成」按钮，即可看到实时预览的 QR 码。
  4. 使用「下载 PNG」或「下载 SVG」保存二维码。

QR 码基础知识

  • 什么是 QR 码？二维条码，能够存储更多信息且具备容错能力。
  • 容错率（Error Correction Level）：L（7%）、M（15%）、Q（25%）、H（30%）。等级越高，可恢复的损坏越多，但码图越密。
  • 版本（Version）：从 1（21×21 模块）到 40（177×177 模块），版本越大容量越高。
  • 数据类型：数字、字母数字、字节、汉字（GBK/UTF-8）等。

代码示例（可折叠）

使用原生 JS 生成 QR 码（基于 VanillaQRCode） 
// 将此脚本放入页面底部或模块中
function generateQR(text, options = {}) {
  const { size = 256, color = '#000', bg = '#fff', eccLevel = 'M' } = options;
  const qr = new VanillaQRCode(text, {
    width: size,
    height: size,
    colorDark: color,
    colorLight: bg,
    correctLevel: VanillaQRCode.CorrectLevel[eccLevel]
  });
  const svg = qr.getSVG();
  document.body.insertAdjacentHTML('beforeend', svg);
}

// 示例：生成一个链接
generateQR('https://example.com', { size: 200, color: '#2563eb', bg: '#f8fafc', eccLevel: 'H' });
使用 Canvas 绘制自定义样式 QR 码 
function drawQRCanvas(text, {size=200, color='#000', bg='#fff', eccLevel='M'} = {}) {
  const qr = new VanillaQRCode(text, {
    width: size,
    height: size,
    colorDark: color,
    colorLight: bg,
    correctLevel: VanillaQRCode.CorrectLevel[eccLevel]
  });
  const canvas = document.createElement('canvas');
  canvas.width = size;
  canvas.height = size;
  const ctx = canvas.getContext('2d');
  // 读取 SVG 并绘制（这里简化为直接使用图像）
  const img = new Image();
  img.onload = () => ctx.drawImage(img, 0, 0, size, size);
  img.src = 'data:image/svg+xml;base64,' + btoa(qr.getSVG());
  document.body.appendChild(canvas);
}

// 示例：绘制带圆角的 QR 码（需后期处理，此处仅展示基本绘制）
drawQRCanvas('Hello World', {size:180, color:'#1e293b', bg:'#f0f4ff'});

常见问题 FAQ（手风琴式）

生成的 QR 码为什么有时候扫不出来？

可能原因包括：

  • 容错率设置过低，导致轻微污损无法恢复。
  • 颜色对比度不足（前景色与背景色相近）。
  • 尺寸过小，扫描设备无法辨识细小模块。

建议：使用中等或高容错率（Q/H），确保前景色深色、背景色浅色，且打印或显示尺寸不小于 2×2 cm。

能否在 QR 码中间嵌入 Logo？

可以。做法是在生成的矩阵中间留出空白区域（通常占总尺寸的 15%-20%），随后将 Logo 图像放置于该区域。注意：

  • Logo 不得覆盖定位图案（三个大方块）。
  • 保持足够的容错率（建议 H 级）以抵抗 Logo 造成的遮挡。

本站的「样式定制」页提供了 Logo 嵌入功能，可直接上传并实时预览。

如何批量生成多个不同内容的 QR 码？

前往 批量生成 页面，支持：

  • 粘贴多行文本，每行生成一个码一个。
  • 上传 CSV（第一列为内容，可选第二列为文件名前缀）。
  • 自定义命名规则（如 prefix_序号_suffix）。

生成后可一次性下载全部 PNG 或打包为 ZIP。

生成的记录会保存在哪里？

所有生成记录默认保存到浏览器的 localStorage，可在 历史记录 页面查看、搜索、导出为 CSV。若需跨设备同步，可使用「设置」页的云同步按钮（示例使用 WebDAV 或自建端点）。
